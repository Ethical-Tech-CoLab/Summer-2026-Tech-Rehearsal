# Summer 2026 Tech Rehearsal — Ethical Tech CoLab

## 🔗 [View the live site →](https://ethical-tech-colab.github.io/Summer-2026-Tech-Rehearsal/)

---

This repository contains the **Summer 2026 Tech Showcase** slide deck for [Ethical Tech CoLab](https://ethicaltechcolab.org) — a fully browser-based, interactive presentation built with HTML and CSS.

The deck introduces the cohort, highlights the season's projects, and serves as a rehearsal tool for the live showcase event. It features:

- A slide-by-slide layout navigable with on-screen arrow buttons or keyboard input
- Team member profiles and project spotlights
- Custom branding with Ethical Tech CoLab's visual identity (Space Mono + Bebas Neue typography, purple/lime color scheme)
- Supporting graphics exported for LinkedIn and Partiful event pages

### Files

| File / Folder | Description |
|---|---|
| `index.html` | Main slide deck (open in any browser) |
| `Public/` | Team photos and the ETC logo used in the deck |
| `LinkedIn-Graphics.html` / `.png` | Social media graphic for LinkedIn |
| `Partiful-Graphics.html` / `.png` | Event graphic for Partiful |
